# Laser Cavity Simulator
Software simulation of laser cavities using ABCD matrix formalism.
The software features a friendly user interface user can interact with by adding optical elements such as mirrors, lens, slabs and optical pumps. The intention of this software is to simulate bidimensional laser cavites using the formalism of the ABCD matrices for pedagogic and experimental purposes.

A few demos are attached below to showcase the software. Full app is shared in this repo and requires Xojo to run: https://www.xojo.com/download/
