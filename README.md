# Laser Cavity Simulator

## Summary
Software simulation of laser cavities using ABCD matrix formalism.
The software features a friendly interactive user interface that contains the option to add optical elements such as mirrors, lens, slabs and optical pumps. The intention of this software is to simulate bidimensional laser cavites using the formalism of the ABCD matrices for pedagogic and experimental purposes.

## Setup
Xojo DEBUG applications require the Xojo IDE to run. Download the latest version from: https://www.xojo.com/download/

This desktop application requires all .png files and the xojo_binary_project to be downloaded and be in the same directory.
